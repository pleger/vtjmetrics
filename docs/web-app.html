Web App

The web app supports two source modes:

  • GitHub URL: repository URL + source path.
  • ZIP upload: local compressed source bundle.

Input Requirements

  • Supported code files: .js, .cjs, .ts (analysis)
  • Import resolution set also includes .jsx, .tsx, .json
  • Source path can be . or any subdirectory (e.g., src)

Visualization Encoding

  • Fan-out segment color: red
  • Fan-in segment color: blue
  • Segment thickness follows ratio over N = fanIn + fanOut for each source node.
