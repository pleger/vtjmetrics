VTJMetrics

VTJMetrics extends JTMetrics with interactive coupling visualization. It supports two runtimes:

  • Chrome extension on GitHub repository pages.
  • Standalone web app (GitHub URL or ZIP source package).
Core metric execution is shared through chrome-extension/browser-engine.js and chrome-extension/metric-src/*.

Main Capabilities

Capability Description
JSON export Download complete JTMetrics output in JSON format.
Coupling visualization Nested circles by precedence and split line encoding for fan-out/fan-in.
Threshold filtering Filter nodes by minimum lines to reduce visual overload.
Interactive exploration Hover details, drag nodes, and metric precedence controls.
