Getting Started

1. Clone

git clone https://github.com/pleger/vtjmetrics.git
cd vtjmetrics

2. Open the web app

Serve the project root with any static server and open /web-app/index.html.

npx serve .
# then visit http://localhost:3000/web-app/index.html

3. Load extension

  1. Open chrome://extensions
  2. Enable Developer mode
  3. Load unpacked -> select chrome-extension/
