Chrome Extension

VTJMetrics extension runs directly on GitHub repository pages.

  • Adds a VTJMetrics tab in repository navigation.
  • Runs metrics in-browser and supports optional GitHub token.
  • Allows interactive coupling visualization and JSON download.
  • Supports package-coupling, file-coupling, cyclic-coupling, temporal-coupling, class-coupling, and function-coupling.
