Architecture

Shared Runtime

Both products reuse the same metric runtime:

  • chrome-extension/browser-engine.js
  • chrome-extension/metric-src/*.metric.js

Execution Paths

Input Entry point
GitHub repository runJTMetricsInBrowser(payload)
ZIP or file map runJTMetricsFromFilesInBrowser(payload)

Design Decision

Metric loading resolves from Chrome runtime URLs in extension mode and from configurable base URL in web mode, reducing duplicated logic.
